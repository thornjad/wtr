# wtr

Get current weather. Doesn't get much simpler than this

## Prerequisites

`wtr` requires `libcurl`, which is usually preinstalled on UNIX-like systems.

## Install

```
make install
```

## Use

```
wtr
wtr Houston
wtr San Fransisco
```

_Note: not case sensitive._

## Why not use Bash?

Because I wanted to use C.
